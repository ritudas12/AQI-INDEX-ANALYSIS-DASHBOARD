🌍 AQI Index Analysis – Power BI Project

 Overview

This project presents an interactive Power BI dashboard that analyzes Air Quality Index (AQI) data. It helps in understanding pollution levels, identifying key pollutants, and observing trends across different cities and time periods.

 Objectives

* Analyze AQI levels across different locations
* Identify the most harmful pollutants
* Track pollution trends over time
* Provide insights for environmental awareness

Features

* City-wise AQI comparison
* Time-based trend analysis (daily/monthly/yearly)
* Pollutant breakdown (PM2.5, PM10, NO2, CO, etc.)
* Interactive filters and slicers
* Visual dashboards with charts and KPIs

 Tools & Technologies

* Power BI
* Data Cleaning & Transformation
* Data Visualization

 Dataset

* The dataset includes AQI readings from multiple locations
* Contains pollutant concentration levels and timestamps
